# Analysis Log

Each subdirectory is a dated, self-contained analysis run. Every directory contains:

- `README.md` — methodology, queries, results, caveats
- `data.json` — raw data snapshot used in the analysis

Naming convention: `YYYY-MM-DD-short-description`

Analyses are referenced by blog posts and should be reproducible from the dataset export.
